## 📖 项目简介

**ArkTS-Learning-Notes** 是一个面向 **HarmonyOS ArkTS 学习与实践** 的个人仓库，用于系统整理课程学习过程中的知识点、课堂案例以及实验作业。

本仓库不仅记录代码实现，也关注学习过程中的思考与总结，力求将零散练习沉淀为一份可持续积累、可随时回顾的 ArkTS 学习笔记。

适合以下读者参考：

- 正在学习 **HarmonyOS / ArkTS** 的初学者
- 需要整理 **课程实验 / 作业代码** 的学生
- 想通过案例逐步理解 ArkTS 语法与逻辑结构的开发者

---

## ✨ 项目特色

- 📘 **系统化整理**：按课程/章节持续更新学习内容
- 🧠 **从基础到实践**：覆盖语法、逻辑结构、控制流程与练习案例
- 🛠️ **作业可运行**：尽量保证代码可读、可运行、可复用
- 📝 **持续沉淀**：不仅保存代码，也记录学习轨迹与经验总结
- 🚀 **适合长期迭代**：后续可逐步扩展为完整的 ArkTS 学习仓库

## 🚀 使用方式

你可以直接克隆本仓库，在本地结合 DevEco Studio 进行学习和运行：

```bash
git clone https://github.com/xiaoluosigithub/ArkTS-Learning-Notes.git
```

然后使用 **DevEco Studio** 打开项目或相关 `.ets` 文件进行查看、调试与运行。
